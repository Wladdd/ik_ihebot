# R2I Bot

> Take submissions from Reddit and post them on Instagram

```
Usage of ./r2i:
  -caption string
    	The post caption (default "#LPT")
  -password string
    	Instagram Password
  -sub string
    	The Subreddit to pull from (default "UnethicalLifeProTips")
  -username string
    	Instagram Username (default "unethicallifeprotips")
```
